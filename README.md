# ⚡ Calculadora de Consumo de Energia

Projeto desenvolvido para estimar o consumo mensal de energia elétrica de aparelhos.

## 🎯 Objetivo
Ajudar o usuário a calcular quanto um aparelho consome de energia por mês.

## 🧪 Tecnologias
- 🐍 Python
- 💻 VS Code
- 🌐 GitHub

## 📊 Fórmula utilizada

consumoMensal = (potencia × horasDia × 30) / 1000

## ▶️ Como executar

1. Abra o terminal
2. Execute o arquivo:

python app.py

3. Digite os dados solicitados

## 💡 Exemplo

Aparelho: Geladeira  
Consumo estimado: 45 kWh/mês  
Custo estimado: R$ 33.75
